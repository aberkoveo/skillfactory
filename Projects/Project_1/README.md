# Проект №1
Учебный проект для [SkillFactory](https://skillfactory.ru/)

Представляет из себя решение 36 задач по данным с портала IMDB


Включает в себя файлы и каталоги:

/README.md - этот файл

/input - папка с источниками данных (.csv)

/[SF-DST] Movies IMBD v3.0.ipynb - рабочий ноутбук, в котором выполняются все вычисления и решения

/my_functions.py - модуль с используемыми в ноутбуке функциями

***
В выводах ячеек содержится статистическая информация по выборке фильмов,

вышедших в период с 2000 по 2015 год
